# AgriBazaar - Smart E-Marketplace for Farmers

## Project Overview

AgriBazaar is an innovative cloud-based e-marketplace platform designed to empower farmers by providing a one-stop solution for buying and selling agricultural products. The platform promotes transparency, efficiency, and profitability by enabling farmers to purchase quality seeds, fertilizers, tools, and other inputs directly from verified sellers. It also allows farmers to upload and sell their farm produce directly to buyers. 

Powered by AI, AgriBazaar offers price forecasting and advanced negotiation features to ensure fair pricing, along with real-time order tracking and logistics support.

---

## Key Features

- **User Registration & Secure Authentication:** Includes email verification and multi-factor authentication.
- **Product Listings:** Farmers can upload produce and agricultural products with descriptions, images, and videos.
- **AI-Powered Price Forecasting:** Recommends prices based on real-time market data and demand-supply trends.

- **Flexible Negotiation and Dispute Resolution:** Buyers and sellers can negotiate prices; admins monitor activity in real time.

- **Responsive UI:** Designed using Bootstrap, HTML, CSS, and JavaScript for ease of use on mobile and desktops.
- **Cloud-Based Storage:** Cloudinary integration for efficient image uploads and management.
- **Robust Backend:** Built on Django with PostgreSQL database (deployed on Neon.tech), and hosted on Vercel platform.
